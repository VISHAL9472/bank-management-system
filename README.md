# 🏦 Bank Management System

A RESTful Bank Management System developed using Java, Spring Boot, Spring Data JPA, Hibernate, and PostgreSQL. The application provides secure banking operations through well-structured REST APIs.

## 🚀 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven
- Postman

## ✨ Features

- Bank Management
- Account Management
- Create Bank
- Update Bank
- Delete Bank
- Create Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Exception Handling
- RESTful APIs

## 📂 Modules

- Bank
- Account
- Address
